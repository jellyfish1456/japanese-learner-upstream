# Japanese Learner App - Project Context

**Project**: 日本語學習網站（Chris 每日日文學習）  
**Tech Stack**: React + TypeScript + Vite + Tailwind CSS v4 + React Router v6  
**Hosted**: GitHub Pages `/japanese-learner-upstream/`  
**Repo**: https://github.com/jellyfish1456/japanese-learner-upstream

## Core Features

1. **Learning Datasets** (N5/N4/N3 詞彙+文法)
   - Vocabulary (vocab-n5.json, vocab-n4.json, vocab-n3.json)
   - Grammar (grammar-n5.json, grammar-n4.json, grammar-n3.json)
   - Create/edit custom datasets

2. **Study Modes**
   - **Learn Mode**: Flashcard-style with swipeable cards + seekable progress bar
   - **Study Mode**: Quiz-based review
   - Each includes completion summary screen

3. **Daily Dialogue** (日常對話)
   - N5/N4/N3 dialogues (15 per level = 45 total)
   - Chat-bubble UI with per-line audio (Web Speech API TTS)
   - Copy-on-click example sentences

4. **Listening Practice** (聽力練習)
   - N5/N4/N3 MCQ: hear Japanese → pick Chinese translation

5. **Verb Conjugation** (動詞變化)
   - 3 verb groups × 10-11 verbs × 11 forms
   - Two views:
     - **按類別**: Group-based (G1 五段 / G2 一段 / G3 不規則)
     - **按活用形**: Day-based (Day 1-7) with detailed Chinese explanations

6. **Settings** (設定)
   - Theme toggle (light/dark)
   - Language (Chinese/English)
   - About page removed

## Key Technical Details

### Furigana (Ruby Text)
- **Data**: `src/lib/furigana-map.json` (501KB, pre-generated by pykakasi)
- **Component**: `src/components/RubyText.tsx` (lazy-loads map via dynamic import)
- **Rendering**: `<ruby>kanji<rt>reading</rt></ruby>` with CSS antialiasing
- **CSS Rules**: `.ruby rt { font-size: 0.55em; -webkit-font-smoothing: antialiased; }`

### Grammar Highlight
- `GrammarHighlight.tsx`: Wraps grammar patterns in 【】 with RubyText
- Modes: "highlight" (blue bg) vs "plain"
- Applied in: LearnCard examples + DialoguePage lines

### Web Speech API (TTS)
- `SpeakButton.tsx`: Click to play Japanese text
- Uses browser `speechSynthesis` with voice `Google 日本語`

### Verb Conjugation Data
- `src/data/verbConjugation.ts`: 11 forms (dict/masu/te/ta/nai/pot/ba/vol/caus/pass/causp)
- G1 (五段): 9 ending patterns + 10 common verbs
- G2 (一段): 1 rule + 10 common verbs  
- G3 (不規則): 2 base forms + 8 compound verbs
- FORM_DETAILS: Day 1-7 explanations with G1/G2/G3 rules

### Data Files
- `src/lib/dialogue-*.json`: Dialogues with sentences, translations, furigana
- `src/lib/listening-*.json`: MCQ items with options, explanations

### Components
- `LearnCard.tsx`: Vocab/Grammar card with similar-grammar comparison
- `DatasetCard.tsx`: Dataset stats card
- `SeekableProgressBar.tsx`: Draggable progress with tick marks
- `Layout.tsx`: Header + nav; home button when not on home page
- `SpeakButton.tsx`: Web Speech API wrapper
- `RubyText.tsx`: Auto-generates furigana from furigana-map

## Common Patterns

**Lazy-load Furigana**:
```tsx
import { loadFuriganaMap, isFuriganaReady } from "../lib/furigana";
```

**Manual Ruby**:
```tsx
<ruby>{text}<rt>{hiragana}</rt></ruby>
```

**Auto Ruby**:
```tsx
<RubyText text={text} />
```

**Audio**:
```tsx
<SpeakButton text={japaneseText} />
```
